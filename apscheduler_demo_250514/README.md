## 2. 실행 방법
1. 의존성 라이브러리 설치
    ```bash
    pip install apscheduler
    ```
   
2. 실행 명령어
    - Windows
       ```bash
       py3 -m apscheduler_demo_250514.main
       ```
    - Linux
       ```bash
      python3 -m ../apscheduler_demo_250514.main
      ```
